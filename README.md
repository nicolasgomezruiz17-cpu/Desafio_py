# Desafio_py

Este proyecto fue desarrollado en un **MacBook Air con Apple Silicon M3**. Debido a que macOS protege su entorno de Python del sistema operativo, no es posible instalar librerías como NumPy directamente con `pip` sin riesgo de interferir con las dependencias del sistema.
 
Por esta razón tuve que subir el proyecto a git hub y usar un **entorno virtual local** para aislar las dependencias del proyecto.
